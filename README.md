# JPG → PDF Converter

A sleek, browser-based tool to merge multiple JPG/PNG images into a single PDF file. **Everything runs locally** — no images are uploaded to any server.

## ✨ Features

- **100% client-side**: All processing happens in your browser. Your images never leave your device.
- **Multiple image support**: Select several JPG or PNG files at once.
- **Flexible page options**:
  - Page sizes: A4, Letter, A3, or "Fit to Image"
  - Orientation: Auto, Portrait, or Landscape
- **Real-time preview**: Thumbnails of processed images appear as they're added to the PDF.
- **Multi-language interface**: Available in 🇬🇧 English, 🇫🇷 French, 🇪🇸 Spanish, 🇵🇹 Portuguese, 🇩🇪 German, and 🇮🇹 Italian.
- **Drag & drop**: Simply drag your images into the drop zone.
- **Download all**: One-click download of the complete PDF.
- **Responsive design**: Works smoothly on desktop and mobile devices.

## 🚀 How to Use

1. **Open the tool** in your web browser.
2. **Add images**: Drag & drop `.jpg`, `.jpeg`, or `.png` files into the drop zone, or click to select them.
3. **Adjust settings** (optional):
   - Choose the desired **page size**.
   - Select the **orientation** (Auto, Portrait, Landscape).
4. Click **"Create PDF"**.
5. Wait for the processing to complete — thumbnails will appear as each image is added.
6. Click **"Download PDF"** to save your merged document.

## 🛠️ Technologies Used

- **HTML5 / CSS3** — Structure and styling
- **Vanilla JavaScript** — Core logic and interactivity
- **[jsPDF](https://github.com/parallax/jsPDF)** — PDF generation library
- **Google Fonts** — Space Mono & Sora typefaces

## 📁 Project Structure

index.html
└── Contains all the code:
├── Inline CSS (styles)
├── HTML markup
└── JavaScript logic (with jsPDF CDN)

No build process, no dependencies to install — just open `index.html` in a browser.
